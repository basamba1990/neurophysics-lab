# Plateforme d'Ingénierie Accélérée par IA (R&D Accelerator)

Ce projet est une plateforme complète pour l'ingénierie et la R&D, intégrant des solveurs basés sur les Physics-Informed Neural Networks (PINN), un Scientific Copilot pour la modernisation de code, et un moteur d'optimisation pour les jumeaux numériques.

## Structure du Projet

Le projet est divisé en deux parties principales :

1.  **backend/** : L'API FastAPI pour la logique métier, les solveurs IA et l'accès aux données.
2.  **frontend/** : L'interface utilisateur React/Vite pour interagir avec la plateforme.

## Démarrage Rapide

Consultez les dossiers `backend/` et `frontend/` pour les instructions de configuration spécifiques.
